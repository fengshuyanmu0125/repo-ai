# 股票行情查询

调用 `get_stock_detail` 工具，获取 $ARGUMENTS 中每只股票的详细信息。

对每只股票，用中文输出以下内容：

**[代码] 公司名**
- 当前价格：$XXX　涨跌：±X.XX%
- 52周区间：低$X ~ 高$X，当前位置 XX%（██████░░░░）
- 分析师：[评级共识]  买入X 持有X 卖出X
- 最新动态：[2-3条新闻的中文摘要，每条1句话]

如用户查询多只（用空格分隔），逐只输出后加一句横向比较。
